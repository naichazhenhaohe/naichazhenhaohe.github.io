어법

连音化的两种情况

在语流中，闭音节跟元音开头的音节相连时，前面这个闭音节的韵尾(韵尾ㅇ,ㅎ除外)成为后面音节的头音，这种语音变化叫做连音化。

韵尾直接成为相连的音节的头音

  1. 깎아 [까까]
  2. 있어요 [이써요]
  3. 꽃이 [꼬치]

闭音节跟元音开头的音节相连，且该元音为无实际意义的助词/连接词，则闭音节韵尾直接成为后面的音节的头音。

韵尾的代表音成为相连的音节的头音

  1. 마없다 [마덥따]
  2. 못 왔어요 [모돠써요]

闭音节跟元音开头的音节相连，且该元音为有实际意义的词如 없다 / 오다，则闭音节韵尾的代表因后移成为后面的音节的头音。

특례

  • 맛있다 [마싣따]
  • 멋있다 [멋싣따]

[명사] + 떼문에 / [동사/형용사/이다] + 기 때문에

表原因、理由

보기

[명사] + 때문에

  1. 비 때문에 등산을 안 갔어요.
  2. 시험 때문에 열심히 공부했어요.

[동사/형용사/이다] + 기 때문에

  1. 비가 오기 때문에 등산을 안 갔어요.
  2. 한국 드라마를 좋아하기 때문에 한국어를 배워요.
  3. 빌리 씨는 미국 사람 이기 때문에 한국말을 못 해요.

제한

  1. 一般不接在 겠 之后，可以接在 았 / 었 / 였 后 （即一般接在过去式之后）
  2. 때문에 之后不能接共动式和命令式。

비교

  1. [동사/형용사] + (으)니까
    表原因，后分句一般为命令式or共动式。
    비가 오나까 우산을 가지고 가세요.
  2. [동사/형용사/이다] + 아/어/여서
    连接词尾，表示原因或者理由
    오늘 나침부터 기침을 많이 해요.목도 아프고요.

못 + [동사] / [동사] + 지 못하다

否定式，表由于没有能力或者外因而不能进行某行为

"不能做…"，"没法做…"

보기

저는 김치를 못 먹어요.(먹지 못해요.)

비가 오니까 등산을 못 가요.(가지 못해요.)

스트레스가 많아서 잠을 못 자요.

특례

못 + 명사 + 하다 时，못 接在 하다 的前面

시간이 없어서 운동 못 해요.(운동하지 못해요.)

단어

작다

형용사 (体积、长度、宽度等)小 ，细 ，矮 ，微

-> 크다 형용사 大 ，高 ，长 ，壮阔 ，重

휴게실

명사 [休憩室] 休息室

하루 종일

명사 全天 ，终日

-> 전일 명사 [全日] 整日 ，整天

좋아지다

자동사 (病情)好起来 ，见好 ，好转 ，变好

시험 공부

명사 [試驗工夫] 准备考试 。备考 。应考复习

꼭

부사 一定 ，肯定 ，必定 ，刚好 ，正好 ，紧紧 ，紧 ，牢牢

가지

의존명사 {수량 당위} 种 ，种类 ，类 ，样 ，个 ，类别 ，项 ，条

조리실

명사 烹饪室

방해

명사 [妨害] 妨害 ，妨碍 ，干扰

금연건물

명사 [禁煙建物] 禁烟建筑物

언제든지

부사 无论什么时候 ，不管什么时间 ，任何时候 ，无论何时 ，不管何时 ，随时 ，[口]哪会儿

오래

부사 很长时间 ，很久 ，好久 ，许久

화가 나다

发火 ，生气

스트레스

명사 [stress] 压力 ，紧张 ，应力

자판기

명사 [自販機] 自动售货机

시끄럽다

형용사 嘈杂 ，喧哗 ，吵闹

주차하다

타동사 [駐車–] 驻车 ，停车 ，泊车
